# my_web

`my_web` is the public interface for the archive.

It does not start as a traditional homepage. The first public surface is a terminal that lets a visitor inspect projects, thoughts, timeline entries, and later richer archive applications.

## Current direction

- Build the terminal as the primary visitor interface.
- Keep the archive data in Markdown and JSON.
- Treat Git as the first content management layer.
- Keep visual experiments separate from the stable public shell.

## v0.1 boundary

The first version supports browsing, opening, searching, and reading public records. Owner editing, login, AI interpretation, and multi-person permissions are intentionally left for later phases.
