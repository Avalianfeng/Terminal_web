# Personal Archive System

The archive starts from `Person`.

A traditional personal site is usually organized as pages: home, about, blog, projects, and contact. This system treats those pages as views over a deeper object.

> 档案从「人」出发，而不是从页面树出发。网站只是展示档案的一种视图。

## Core object

The highest-level object is the person.

Projects, thoughts, documents, timeline events, media, and relations are attached to that person. The website is only one way to display the archive.

### First interface

The first interface is a **terminal**.

This is not meant as decoration. The terminal is the browsing method, the command surface, and the first archive shell.

---

```text
visitor@archive:~$ open thoughts/archive-system
```
